## 질문

## 확인 내용

공식 API로 안정적인 것은 실거래가입니다. 국토교통부 공공데이터포털 API는 무료 REST/XML 형태로 제공되며, 법정동 코드와 계약년월 기준 조회에 적합합니다.

현재 올라온 매물 호가는 공공 공식 API로 보기 어렵습니다. 네이버부동산/직방/다방 계열은 제휴, 상용 API, 또는 비공식 내부 API/크롤링 영역입니다.

## 국토부 API 검증

기준일: 2026-06-14 15시대 KST

스크린샷의 `일반 인증키`를 일회성 환경변수로 사용해 호출했습니다. 키 원문은 저장하지 않았습니다.

### 연립다세대 매매

- Endpoint: `https://apis.data.go.kr/1613000/RTMSDataSvcRHTrade/getRTMSDataSvcRHTrade`
- 테스트: `LAWD_CD=11215`, `DEAL_YMD=202501`
- 결과: 성공
- 샘플:
  - `2025-01-22 구의동 힐알펜시아 56.22㎡ 4층 66,000만원`
  - `2025-01-31 중곡동 나경타운 53.69㎡ 6층 24,000만원`
  - `2025-01-25 중곡동 햇빛촌빌 27.95㎡ 4층 23,500만원`

### 아파트 매매

- Endpoint: `https://apis.data.go.kr/1613000/RTMSDataSvcAptTrade/getRTMSDataSvcAptTrade`
- 테스트: `LAWD_CD=11215`, `DEAL_YMD=202501`
- 결과: 성공. 단, 같은 키/파라미터에서도 간헐적으로 `HTTP 403 Forbidden`이 발생합니다.
- 샘플:
  - `2025-01-25 자양동 대동 59.54㎡ 18층 96,500만원`
  - `2025-01-20 광장동 워커힐 166.91㎡ 3층 244,900만원`
- 해석: 아파트 API 활용승인은 되어 있으나, API 서버 응답이 불안정하므로 retry/backoff가 필요합니다.
