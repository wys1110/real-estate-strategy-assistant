# 현재 스냅샷

기준일: 2026-06-14

## 질문

한국 주택 매물 가격을 API로 가져올 수 있는지, 그리고 테스트로 자양동 빌라 현재 매물을 가져올 수 있는지 확인했습니다.

## 확인 내용

공식 API로 안정적인 것은 실거래가입니다. 국토교통부 공공데이터포털 API는 무료 REST/XML 형태로 제공되며, 법정동 코드와 계약년월 기준 조회에 적합합니다.

현재 올라온 매물 호가는 공공 공식 API로 보기 어렵습니다. 네이버부동산/직방/다방 계열은 제휴, 상용 API, 또는 비공식 내부 API/크롤링 영역입니다.

## 자양동 테스트 결과

대상:

- 서울특별시 광진구 자양동
- region code: `1121510500`
- 거래: 매매
- 유형: 빌라/연립/다세대

결과:

- 네이버부동산 내부 JSON: `429 TOO_MANY_REQUESTS`
- 부동산뱅크 HTML: 접근 가능
- 부동산뱅크 첫 페이지: 전체 30건 파싱
- 빌라/연립/다세대 계열: 16건 파싱
